R - count the world again
T - save changes
W, A, S, D - move around the map
F1-F6 - select application resolution
F8 - full screen
In texture mapping mode:
LMB - texture mapping
Shift + LMB - texture overlay with a large scale
Ctrl + LMB - replace the texture with a similar one
Rotating the mouse wheel in the texture palette area - selecting a texture
In object editing mode:
Shift + drag object - move a fixed distance
Ctrl + move object - copy object
Del - deleting an object
In NPC editing mode:
Ctrl + move NPC - copy NPC
Del - delete NPC


R - считать мир заново
T - сохранить изменения
W, A, S, D - перемещение по карте
F1-F6 - выбор разрешения приложения
F8 - во весь экран
В режиме наложения текстур:
	ЛКМ - наложение текстуры
	Shift + ЛКМ - наложение текстуры с большим масштабом
	Ctrl + ЛКМ - замена текстуры на аналогичеую
	Вращение колесика мыши в области палитры текстур - выбор текстуры
В режиме редактирования объектов:
	Shift + перетаскивание объекта - перемещение на фиксированное расстояние
	Ctrl + перемещение объекта - копирование объекта
	Del - удаление объекта
В режиме редактирования НПС:
	Ctrl + перемещение НПС- копирование НПС
	Del - удаление НПС
